croatian-zip-codes
==================

Croatian zip codes in json format sorted by counties.

Example:
```
[{
    "name": "Zagrebačka",
    "slug": "zagrebacka",
    "cities": [
    {
        "name": "Gornji Stupnik",
        "slug": "gornji-stupnik",
        "zip_code": "10255"
    }, {
        "name": "Prigorje Brdovečko",
        "slug": "prigorje-brdovecko",
        "zip_code": "10291"
    }
}]
```
